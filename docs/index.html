Polly

Polly is a light-weight TypeScript based AI coding CLI built for Pollinations. Powered by Pollinations BYOP and built for the GPT 5.2 model, while supports other Pollinations models.

Features:

  • Terminal Use
  • Model Selector
  • BYOP
  • Easy to Fork
  • Web Support (use /web)

The only AI CLI built for Pollinations.

Install

$ git clone https://github.com/gamerc0der/polly
$ cd polly/cli
$ npm install -g .
